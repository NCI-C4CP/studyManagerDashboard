EpiSphere Dashboards

Dashboards for multiple Episphere projects. Code at github.com/episphere/dashboard. To make experimenting with shared libraries easier, the different dashboards are developed in subdirectories of the dashboard repository. This is just an idea, it makes senso for more formal team-based development to fall back to the convetional gitflow model.
